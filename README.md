## Description

루틴더는 스티커로 습관을 관리하고 트래킹하는 웹앱 서비스입니다. 본 레포는 루틴더 서비스의 클라이언트 코드를 위한 레포입니다. 

## Installation

기본적인 라이브러리 설치
```bash
$ npm install
```

## 실행

```
$ npm start
```

## Storybook 실행

```
$ npm run storybook
```
